# Spotify Power BI Mini Project 🎧

## Phase 1 - Task 1: Data Understanding & Profiling

### Dataset Overview
The dataset used in this project contains Spotify track information for music analysis and visualization.

### Structure
- Total Columns: 21
- Total Rows: 999+
- Missing Values: None (clean dataset)
- Data Quality: High consistency across all features

### Key Columns
- track_name
- artist
- genre
- danceability
- energy
- valence
- tempo
- popularity

### Initial Observations
- Dataset is already well-structured and clean
- No null values detected across any column
- Numerical features are suitable for analysis and visualization
- Categorical features (genre, artist) are ready for grouping
